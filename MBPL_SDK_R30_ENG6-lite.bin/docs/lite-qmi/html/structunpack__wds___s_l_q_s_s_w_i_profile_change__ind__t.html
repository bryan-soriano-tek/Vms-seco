Detailed Description

This structure contains the Profile Change Indication Information

Parameters
type
  • Type of profile, valid for TLV 0x01 0x00 - 3GPP 0x01 - 3GPP2
index
  • Index identifying the profile, valid for TLV 0x01 1-16 (9x15 and before) or 1-24 (9x30 and onwards) valid for 3GPP profile type 101-106 valid for 3GPP2
source
  • Soruce of change, , valid for TLV 0x10 0 - Modem (AT) 1 - Host (QMI client) 2 - OMADM
Tlvresult
  • TLV present flag, TURE if presented, otherwise FALSE
ParamPresenceMask
  • bitmask representation to indicate valid parameters.
