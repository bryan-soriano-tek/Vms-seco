Detailed Description

SO Mask

  • SO mask to indicate the service option or type of application.
    • An SO mask value of zero indicates that this field is ignored.
    • Values:
      • 0x00 - DONT_CARE
    • CDMA 1X SO mask:
      • 0x01 - CDMA_1X_IS95
      • 0x02 - CDMA_1X_IS2000
      • 0x04 - CDMA_1X_IS2000_REL_A
    • CDMA EV-DO Rev 0 SO mask:
      • 0x01 - DPA
    • CDMA EV-DO Rev A SO mask:
      • 0x01 - DPA
      • 0x02 - MFPA
      • 0x04 - EMPA
      • 0x08 - EMPA_EHRPD
    • CDMA EV-DO Rev B SO mask:
      • 0x01 - DPA
      • 0x02 - MFPA
      • 0x04 - EMPA
      • 0x08 - EMPA_EHRPD
      • 0x10 - MMPA
      • 0x20 - MMPA_EHRPD

RAT Mask

  • RAT mask to indicate the type of technology. A RAT mask value of zero indicates that this field is ignored.
    Values:
    • 0x00 - DONT_CARE
    • 0x8000 - NULL_BEARER
  • CDMA RAT mask:
    • 0x01 - CDMA_1X
    • 0x02 - EVDO_REV0
    • 0x04 - EVDO_REVA
    • 0x08 - EVDO_REVB
    • 0x10 - EHRPD
    • 0x20 - FMC
  • UMTS RAT mask:
    • 0x01 - WCDMA
    • 0x02 - GPRS
    • 0x04 - HSDPA
    • 0x08 - HSUPA
    • 0x10 - EDGE
    • 0x20 - LTE
    • 0x40 - HSDPA+
    • 0x80 - DC_HSDPA+
    • 0x100 - 64_QAM
    • 0x200 - TD-SCDMA
