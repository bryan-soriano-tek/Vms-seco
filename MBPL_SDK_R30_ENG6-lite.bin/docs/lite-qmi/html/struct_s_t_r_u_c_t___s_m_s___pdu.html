Parameters
sms_Ref_Num
  • SMS message regference number
message_Status
  • Message status
    • Values:
      • 0 - New
      • 1 - Read
      • 2 - Draft
      • 3 - Sent
message_Encoding
  • Message encoding type
    • Values:
      • 1 - 7bit Encoding
      • 2 - 8bit Encoding
      • 3 - UCS2 Encoding
fragmented_message
  • Concatinated SMS
  • 0: not a concatinated SMS
  • 1: not a concatinated SMS
total_fragments
  • SMS total fragments count
fragment_index
  • SMS fragment index
timestamp
  • timestamp of the SMS: YYMMDDHHMMSSTZ where
    • timestamp[0] YY: year (year-2000, add 2000 to get the year)
    • timestamp[1] MM: month
    • timestamp[2] DD: day
    • timestamp[3] HH: hour
    • timestamp[4] MM: minute
    • timestamp[5] SS: second
    • timestamp[6] TZ: timezone All values are in decimal. The timezone is in relation to GMT, one unit is equal to 15 minutes and the MSB indicates a negative value
moAddress
  • Mobile originated address
scAddress
  • Service center address
pMessage
  • SMS text message
