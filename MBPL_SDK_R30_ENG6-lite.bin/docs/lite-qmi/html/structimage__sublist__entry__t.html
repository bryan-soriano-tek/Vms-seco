Detailed Description

This structure contains device stored image sublist entry.

Parameters
storage_indexIndex in storage where the image is located. If not relevant for the image type, this field is set to 0xFF.
failure_countNumber of consecutive write attempts to this storage index that have failed. If not relevant for the image type, this field is set to 0xFF.
image_idImage unique identifier
build_id_lenBuild identifier length
lru_rankLeast Recently Used rank for the firmware image. This field is not relevant to PRI image and set to 0xFF.
switch_failureNumber of consecutive firmware image switching failures using this storage index. If not relevant for the image type or not supported, this field is set to 0xFF.
dld_sourceSource info on how the image was downloaded to device
  • 0x0: Local download
  • 0x1: FOTA download
  • 0xFF: Not applicable This field is only relevant to firmware image. For other image types, such as PRI images, this field is set to 0xFF.
