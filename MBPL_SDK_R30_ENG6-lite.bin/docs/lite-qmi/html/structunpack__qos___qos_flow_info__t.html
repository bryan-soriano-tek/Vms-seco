Detailed Description

Structure with QoS flow details.
Please check is_<Param_Name>_Available field for presence of optional parameters

Parameters
QFlowState
  • QoS flow state information, please check unpack_qos_QosFlowInfoState_t for more information
is_TxQFlowGranted_Available
  • TRUE if optional TxQFlowGranted is available
TxQFlowGranted
  • The Tx Qos flow granted, please check unpack_qos_swiQosFlow_t for more information
is_RxQFlowGranted_Available
  • TRUE if optional RxQFlowGranted is available
RxQFlowGranted
  • The Rx Qos flow granted, please check unpack_qos_swiQosFlow_t for more information
NumTxFilters
  • Number of Tx filters available
TxQFilter
  • The Tx Qos filter, please check unpack_qos_swiQosFilter_t for more information
  • See LITEQMI_MAX_QOS_FILTERS for more information
NumRxFilters
  • Number of Tx filters available
RxQFilter
  • The Rx Qos filter, please check unpack_qos_swiQosFilter_t for more information
  • See LITEQMI_MAX_QOS_FILTERS for more information
BearerID
  • The bearer ID
  • Bearer ID or Radio Link Protocol (RLP) ID of the activated flow.
  • Valid Values - 0 to 16
  • 0xFF - Invalid value.
is_QosFlowType_Available
  • TRUE if optional QosFlowType is available
QosFlowType
  • QOS Flow Type
  • Indicates whether the QoS flow type is network-initiated or UE-initiated.
  • 0x00 - UE-initiated QoS flow
  • 0x01 - Network-initiated QoS flow
is_BearerID_Available
  • TRUE if optional BearerID is available
is_FcSeqNum_Available
  • TRUE if optional FcSeqNum is available
FcSeqNum
  • Flow Control Sequence Number
  • The sequence number of each flow enable and disable event.
  • Whenever the flow is disabled, the sequence number is incremented.
  • The sequence number remain the same when the flow is enabled.
