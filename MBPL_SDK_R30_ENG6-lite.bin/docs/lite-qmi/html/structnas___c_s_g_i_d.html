Detailed Description

Contain the CSGID.

Parameters
mcc
  • MCC value. Range 0 to 999
mnc
  • MNC value. Range 0 to 999
mncPcsDigits
  • TRUE - MNC is a three-digit value; e.g., a reported value of 90 corresponds to an MNC value of 090
  • FALSE - MNC is a two-digit value; e.g., a reported value of 90 corresponds to an MNC value of 90
id
  • Closed subscriber group identifier.
rat
  • Radio interface technology of the CSG network. Values:
    • 0x04 - RADIO_IF_GSM - GSM
    • 0x05 - RADIO_IF_UMTS - UMTS
    • 0x08 - RADIO_IF_LTE - LTE
    • 0x09 - RADIO_IF_TDSCDMA - TDS
    • 0x0C - RADIO_IF_NR5G - NR5G
