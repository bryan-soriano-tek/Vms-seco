Detailed Description

lite-qmi library is a simplified wrapper around QMI objects. QMI (Qualcomm MSM Interface) APIs are provided to facilitate the following:

  • Packing QMI packets from fixed user parameter structs
  • Parsing/Unpacking QMI packets to fixed user parameter structs
