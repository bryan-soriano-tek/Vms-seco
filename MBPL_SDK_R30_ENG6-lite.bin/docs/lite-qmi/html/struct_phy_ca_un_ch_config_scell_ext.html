Detailed Description

This structure contains the parameters for Unchanged Configured Scell Extended Information

Parameters

      • NAS_LTE_CPHY_CA_BW_NRB_75(0x04) - 15
      • NAS_LTE_CPHY_CA_BW_NRB_100(0x05) - 20
