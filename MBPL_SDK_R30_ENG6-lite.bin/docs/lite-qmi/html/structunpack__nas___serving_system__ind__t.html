Detailed Description

This structure contains unpack set serving system callback indication parameters.

Parameters

    • See nas_ServingSystemInfo for more details
  • Bit to check in ParamPresenceMask - 1

    • Values:
      • 0x00 - ROAMING_IND_ON - Roaming
      • 0x01 - ROAMING_IND_OFF - Home
      • 0x02 - ROAMING_IND_FLASHING - Flashing
      • 0x03 and above - Operator-defined values
  • Bit to check in ParamPresenceMask - 0x10

DefaultRoamingIndicator
  • Roaming indicator.
    • Values:
      • 0x00 - ROAMING_IND_ON - Roaming
      • 0x01 - ROAMING_IND_OFF - Home Values from 2 onward are applicable only for 3GPP2. Refer to 3GPP2 C.R1001-F for the meanings of these values.
  • Bit to check in ParamPresenceMask - 0x16
