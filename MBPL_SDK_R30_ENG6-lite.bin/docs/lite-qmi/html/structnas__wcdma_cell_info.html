Detailed Description

This structure contains information about the WCDMA Cell.

Parameters
psc
  • Primary scrambling code.
  • Range: 0 to 511.
cpich_rscp
  • Absolute power level (in 1/10 dBm) of the common pilot channel as received by the UE.
  • Range: -120.0 dBm to -25.0 dBm
cpich_ecno
  • CPICH Ec/No; ratio (in 1/10 dB) of the received energy per PN chip for the CPICH to the total received power spectral density at the UE antenna connector.
  • Range: -50.0 dB to 0.
srxlev
  • Cell selection Rx level (Srxlev) value.
  • Range: -128 to 128.
  • This field is only valid when ue_in_idle is TRUE.
