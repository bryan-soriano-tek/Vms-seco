Detailed Description

This structure contains the DSD Register ui changes request parameters.

Parameters
pReport_ui_changes
  • Register/Deregister for UI information changes. Values: 0 - Do not report 1 - Report UI change (default).
pSuppress_so_change
  • Suppress SO mask. Values:
    • 0 - Do not suppress only SO mask changes (Default).
    • 1 - Suppress SO mask change. The indication is not generated only if SO mask changes from the last indication. Ex: Indiation is sent when technology, rat_value, or ui_mask change from the last indication.
PSuppress_null_bearer_reason
  • Suppress null bearer reason. Values:
    • 0 - Do not suppress only null bearer reason changes (Default).
    • 1 - Suppress null bearer reason change. The indication is not generated only if null bearer reason changes from the last indication.
